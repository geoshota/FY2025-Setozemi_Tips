# 地理情報系

## [国土数値情報](https://nlftp.mlit.go.jp/ksj/)

国土数値情報から取得することができるデータには，**「非商用」**，**「商用可」**，**「CC_BY_4.0」**，**「 CC_BY_4.0（一部制限）」**，など，利用方法に規則が付与されている。したがって，分析等で利用する場合は，規則に十分留意すること。

なお，データの内容によっては，更新が入る場合がある（本記事作成日時：2025年4月1日）。そのため，掲載リンクが適切に反映されない場合がある。その場合は，国土数値情報のHPから再度，希望するデータのページに飛び，取得されたい。
